# User guide

1. **First import (recommended)**: Open **`/admin`**, upload the XLSX (`Departments`, `Positions`, `Managers`, `Employees`). This resets documents and master data. If the server sets `ADMIN_IMPORT_TOKEN`, enter it on that page.
2. **Sign in**: Use a **login name** from the **Login** column (no password), e.g. `user1` or `ruk1`.
3. **Library**: Create documents, open the editor, use export or approval as needed.
4. **Language**: Use **Language** in the header to switch between English (default) and Chinese.
