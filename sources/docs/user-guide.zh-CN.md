# 用户指南

1. **首次导入（推荐）**：打开 **`/admin`**，上传 XLSX 文件（`Departments`、`Positions`、`Managers`、`Employees`）。这将重置文档和主数据。如果服务器设置了 `ADMIN_IMPORT_TOKEN`，请在该页面输入。
2. **登录**：使用 **Login** 列中的**登录名**（无需密码），例如 `user1` 或 `ruk1`。
3. **文库**：创建文档，打开编辑器，根据需要导出或审批。
4. **语言**：使用顶栏的 **Language** 在英文（默认）和中文之间切换。
